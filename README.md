Sudden-Death-ChromeExtention
============================

Sudden-Death(http://sudden-death.papix.net/) Chrome Extention


インストール
============================
crx/Sudden-Death-ChromeExtention.crx をchromeの拡張機能のページにドラッグ・アンド・ドロップしてください。


つかいかた
============================
テキストを選択後右クリックし「突然の...」をクリック
